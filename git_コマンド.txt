Gitコマンド一覧表

git init 開いているディレクトリに.gitというディレクトリを作成する　この中にGitを管理する情報が入っている 
git clone <リポジトリURL> 他のリポジトリを自身のリポジトリとして複製し自分の作業用のリポジトリにすることができる
git fetch 他のリポジトリにあるブランチを最新状態の物をを自分のブランチに反映させる
git branch 　現在のブランチ名の前に（*ブランチ名）で表示され、リポジトリにある全てのブランチが表示される
git branch -D "ブランチ名"　ブランチを強制的に消去する
git checkout 現在のブランチを移動する
git checkout -b 新しくブランチを作成する
git add -A 変更したファイルをステージングエリア（リポジトリにコミットするファイルの置き場所）に追加する
git commit -m "メッセージ"　ステージングエリアに追加されたファイルを委託します　メッセージは変更点を簡潔に記載します
git push origin "ブランチ名" 自身のリポジトリに変更点を送信します
